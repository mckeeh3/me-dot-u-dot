About this App

Me-Dot-U-Dot is an Akka-powered application that explores self-learning AI agents within a two-player board game environment. The backend uses event-sourced entities, views, and agent components to manage game state, player data, and agent playbooks. The UI provides a clean, modern experience to select or create players, start games, and observe gameplay or agent decisions.

Game Page

  • Use the navigation button to switch between pages.
  • Select or create Player 1 and Player 2. Agents require a model selection.
  • Choose a game level after both players are selected, then begin the game.
  • During play, the board shows available moves and current status. Agent turns are automatic.
  • You can cancel a game, or reset after it completes.
  • For agent players, open the journal viewer to read the agent's playbook entries.

Playbook Viewer

  • Select an agent to view their playbook journal and navigate between entries.
  • Entries display metadata and instructions used to guide agent behavior.
  • The viewer shares the same visual style and dropdown components as the game page.
